# 数组的高级用法

1.将一维数组 遍历成二维数组

```js
   categorysArr () {
          const {categorys} = this
          const arr = []
          let smallArr = []
          categorys.forEach(c => {
            // 将全新的小数组保存到大数组中
            if(smallArr.length===0) {
              arr.push(smallArr)
            }

            // 将当前分类保存到小数组
            smallArr.push(c)

            // 如果刚好满了, 为后面创建一个新的小数组
            if(smallArr.length===8) {
              smallArr = []
            }
          })
          return arr
        }
      }
```
